# zhangwei89.github.io

个人主页 / GitHub Pages 静态站点。所有内容与样式都在仓库根目录的 `index.html` 中。

## 本地预览

直接打开 `index.html`，或在该目录下运行任意静态服务器，例如：

```bash
python -m http.server 8000
# 浏览器访问 http://localhost:8000
```

## 发布

1. 推送到 `main` 分支。
2. Settings -> Pages -> Deploy from a branch -> `main` / `/` (root)。
3. 几分钟后访问 `https://<username>.github.io/`。

## 内容来源

指南清单来自 GitHub `?tab=repositories`。新增或删除仓库时，编辑 `index.html` 里的 `<article class="category">` 块即可。
